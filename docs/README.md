# docs

このディレクトリは、`senior-ai-chat` の要件定義・設計文書を置く場所です。

初期段階では実装コードよりも、目的、対象利用者、運用、安全性、将来の設計判断を明文化することを優先します。

## 文書一覧

- `01_requirements.md`: MVP要件定義の初期版
- `02_screen_design.md`: 画面設計の初期メモ
- `03_data_model.md`: データモデル初期案
- `04_authentication_design.md`: パスキー認証設計の初期メモ
- `05_realtime_chat_design.md`: リアルタイムチャット設計の初期メモ
- `06_admin_design.md`: 管理者機能設計の初期メモ
- `07_mvp_development_plan.md`: MVP開発計画
- `decisions/`: 設計判断や決定事項の記録

## 注意事項

- 実在の同期生情報・個人情報は絶対に入れない
- 未確定事項は確定事項として書かず、仮説または未決定事項として扱う
- GitHub公開を前提に、外部に出せない情報を含めない
- 実装に進む前に、関連する要件・設計文書を更新する
