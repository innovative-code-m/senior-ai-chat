# src

このディレクトリは、将来の実装コードを置く場所です。

現在の初期化段階では、Reactアプリ生成、ASP.NET Coreプロジェクト生成、パッケージ追加は行いません。

## ディレクトリ

- `frontend/`: React + TypeScript のフロントエンド実装予定地
- `backend/`: ASP.NET Core Web API + SignalR のバックエンド実装予定地

## 注意事項

- 実装開始前に関連する `docs/` を更新する
- パスワード認証を追加しない
- MVP範囲外の機能を勝手に追加しない
- 実在の個人情報をサンプルデータとして含めない
