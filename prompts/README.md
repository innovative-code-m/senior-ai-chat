# prompts

このディレクトリは、AIエージェントに作業を依頼するためのプロンプトや、検討に使った指示文を保存する場所です。

## 用途

- 初期化プロンプト
- 設計レビュー用プロンプト
- 調査依頼プロンプト
- 実装前の要件整理プロンプト
- 利用者向け説明文の草案作成プロンプト

## ディレクトリ

- `init/`: リポジトリ初期化や設計初期化に使うプロンプト
- `exec/`: 実装や検証など、作業実行時に使うプロンプト

## 方針

- プロンプトには実在する参加者の個人情報を含めない
- 実験上の仮説と確定事項を分けて書く
- 実装を依頼するプロンプトでは、対象ファイルと期待する変更範囲を明確にする
- 安全性、プライバシー、高齢者向けの読みやすさを明示する
